# PilotWrt

> Modern Android client for OpenWrt.

PilotWrt is a modern Android application designed for managing OpenWrt, ImmortalWrt, and iStoreOS routers.

## Features

- 📊 Real-time Dashboard
- 📱 Material Design 3
- 🔌 Plugin Management
- 🖥 Device Management
- 🚀 OpenClash Control
- 📈 Live Traffic Monitor
- 🔒 Secure HTTPS & SSH
- 🌙 Dark Mode
